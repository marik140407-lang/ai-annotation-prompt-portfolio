# ai-annotation-prompt-portfolio
Mini portfolio: data annotation, AI evaluation, and prompt engineering samples.
This repository contains a small collection of my practical work samples as an AI data annotator, AI evaluator, and prompt specialist.

**Highlights:**
- Sentiment and intent labeling examples for English texts.
- Annotation guidelines and feedback to human annotators.
- Evaluation of large language model (LLM) answers (e.g., VPN explanations).
- System prompts for an English writing assistant (B1–B2 learners) and an AI travel planner for older budget travelers.
